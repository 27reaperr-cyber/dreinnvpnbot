## Особенности

### Для разработчиков

#### 1. Использование в HTML сообщениях
```javascript
// Способ 1: с функцией em()
await tg("sendMessage", {
  chat_id: chatId,
  text: `${em("🎉", "celebrate")} <b>Поздравляем!</b>`,
  parse_mode: "HTML"
});

// Способ 2: жесткий тег
text: '<tg-emoji emoji-id="6041731551845159060">🎉</tg-emoji> Поздравляем!'
```

```javascript
// Способ 1: с функцией createEmojiBtnInline()
[createEmojiBtnInline("Купить VPN", "v:buy", "money")]

// Способ 2: ручно
{
  text: "Купить VPN",
  callback_data: "v:buy",
  icon_custom_emoji_id: E.money
}
```

## Примечания

1. **Поддержка Premium**: Эмодзи работают для всех пользователей Telegram, но выглядят красивее для пользователей с Telegram Premium
2. **Обратная совместимость**: Все исходные эмодзи сохранены как fallback
3. **Производительность**: Использование эмодзи не влияет на производительность
4. **Безопасность**: ID эмодзи - это публичные данные, никакой опасности нет

## Как добавить новый эмодзи

1. Открыть файл `emojis.js`
2. Добавить новый ID в объект `module.exports`:
```javascript
myNewEmoji: "YOUR_EMOJI_ID",
```
3. Использовать в коде:
```javascript
button: createEmojiBtnInline("Текст", "callback", "myNewEmoji")
```
