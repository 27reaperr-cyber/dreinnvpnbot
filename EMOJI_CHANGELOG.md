## 🚀 Как использовать

### Для новых функций/текстов в сообщениях

```javascript
// Вместо обычного текста:
"📨 Рассылка"

// Используйте:
`${em("📣", "megaphone")} Рассылка`
```

### Для кнопок

```javascript
// Вместо обычной кнопки:
{text: "💳 Купить", callback_data: "v:buy"}

// Используйте:
createEmojiBtnInline("Купить", "v:buy", "money")
```

## 📝 Примеры расширения

### Добавить новый эмодзи

1. В `emojis.js`:
```javascript
myAwesomeEmoji: "123456789012345678",  // 🆕 Новый эмодзи
```

2. В `bot.js`:
```javascript
// В HTML сообщении
text: `${em("🆕", "myAwesomeEmoji")} Новая функция!`

// В кнопке
createEmojiBtnInline("Новая", "v:new", "myAwesomeEmoji")
```

## ⚙️ Известные особенности

1. **Отображение**: Правильно отображаются на свежих версиях Telegram
2. **Fallback**: Если эмодзи не поддерживается, показывается исходный эмодзи
3. **HTML парсинг**: Обязательно использовать `parse_mode: "HTML"`
4. **ID как строки**: ID эмодзи всегда должны быть строками
